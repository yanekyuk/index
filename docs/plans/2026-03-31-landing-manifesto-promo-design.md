# Landing Manifesto Promo Design

**Date:** 2026-03-31

**Goal:** Replace the current `They're closer than you think` landing-page section with a full-width promo banner that reuses the `found-in-translation-6` hero visual and drives users to that page.

## Approved Direction

- Remove the existing split layout with the animated node graph, long-form copy, and external `Read the story` link.
- Insert a full-width promo block in the same location on the landing page.
- Use the `found-in-translation-6` hero image treatment as the background visual language.
- Keep the promo content short: label, headline, teaser copy, and a clear internal CTA.
- Link the CTA to `/found-in-translation-6`.

## Content Shape

- Label: `Found in Translation`
- Headline: short and high contrast, positioned over the visual
- Body copy: 1-2 short sentences framing the page as a manifesto / visual essay
- CTA: primary button that clearly sends users into the full page

## Visual Notes

- Full-bleed background image with a dark overlay for readability
- Light grid / framing accents borrowed from the hero treatment
- Desktop layout should feel like a banner, not a second article section
- Mobile layout should keep the text and CTA readable without requiring precise image cropping

## Cleanup

- Remove the old section's unused animated graph ref, effect, and CSS helpers once the promo block replaces it
